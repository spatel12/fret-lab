# FretLab

A Flutter desktop app for visualizing and practicing guitar scales across any tuning.

<img width="1512" height="946" alt="image" src="https://github.com/user-attachments/assets/4f5346b5-393c-4350-aeb0-1b984ad244a9" />

## Features

- **16 scales** — Major, Minor, Pentatonic, Blues, all 7 modes, and exotic scales (Whole Tone, Diminished, Hungarian Minor)
- **8 tuning presets** — Standard, Drop D, Drop C, C Standard, Open G, Open D, DADGAD, Half Step Down
- **Per-string tuning adjustment** — nudge individual strings up or down from any preset
- **Scale overlay** — layer two scales on the fretboard simultaneously with distinct colors to see how they relate
- **Display modes** — toggle between note names, intervals, and scale degrees
- **Audio playback** — tap any note on the fretboard to hear it
- **Dark/light mode**
- **24 frets** rendered with a custom fretboard painter

## Getting Started

```bash
# Install Flutter: https://docs.flutter.dev/get-started/install
flutter pub get
flutter run -d macos
```
